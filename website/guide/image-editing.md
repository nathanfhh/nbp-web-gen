# 圖片編輯

圖片編輯模式讓你上傳參考圖片，進行風格轉換或內容修改。

## 基本用法

1. 選擇「圖片編輯」模式
2. 上傳一張或多張參考圖片
3. 描述你想要的修改
4. 點擊「生成」

## 編輯類型

### 風格轉換

將照片轉換成不同的藝術風格：

```
將這張照片轉換成吉卜力動畫風格
```

### 內容修改

修改圖片中的特定元素：

```
把背景換成海邊夕陽
將人物的衣服改成紅色
```

### 圖片融合

結合多張參考圖片的元素：

```
使用第一張圖的角色，放入第二張圖的場景中
```

## 參考圖片

### 支援格式

- JPG、PNG、WebP
- 最多 5 張參考圖片

### 多圖上傳

你可以上傳多張參考圖片，在提示詞中指定如何使用：

```
參考圖 1 的構圖，使用參考圖 2 的色彩風格
```

## 手繪模式

如果你想用手繪草稿作為參考：

1. 點擊「手繪」按鈕開啟畫布
2. 繪製簡單的草稿或輪廓
3. AI 會根據你的草稿生成完整圖片

### 畫布工具

| 工具 | 說明 |
|------|------|
| 畫筆 | 自由繪製（快速鍵：B） |
| 橡皮擦 | 擦除錯誤（快速鍵：E） |
| 平移 | 拖曳畫布檢視不同區域 |
| 顏色選擇器 | 選擇畫筆顏色，支援預設色票和自訂顏色 |
| 筆刷大小 | 調整線條粗細（2-50px） |
| 復原/重做 | 修正操作（快速鍵：Ctrl/Cmd+Z、Ctrl/Cmd+Shift+Z） |
| 清空畫布 | 清除所有繪製內容 |

### 畫布選項

| 選項 | 說明 |
|------|------|
| 畫質 | 輸出解析度：1K、2K、4K |
| 寬高比 | 畫布比例：1:1、16:9、9:16、4:3、3:4 |

### 縮放控制

- **放大**：檢視細節
- **縮小**：檢視全貌
- **重設縮放**：回到 100% 縮放

### 鍵盤快速鍵

| 按鍵 | 功能 |
|------|------|
| B | 切換畫筆工具 |
| E | 切換橡皮擦 |
| Ctrl/Cmd + Z | 復原 |
| Ctrl/Cmd + Shift + Z | 重做 |
| ESC | 取消編輯 |
| Enter | 完成並保存 |

::: tip 工具欄可拖曳
工具欄可以被拖曳到畫面上任何位置，方便你在需要更多畫布空間時調整。
:::
