# 影片模式

<TryItButton mode="video" prompt="一隻蝴蝶緩緩從花朵上飛起，陽光透過翅膀形成彩虹光暈，背景是模糊的花園" />

影片模式使用 Google Veo 3.1 API 生成 AI 影片。

## 基本用法

1. 選擇「影片」模式
2. 選擇生成方式
3. 輸入影片描述
4. 設定影片參數
5. 點擊「生成」

## 生成方式

影片模式提供四種生成方式：

### 文字生成

最基本的方式，純粹用文字描述來生成影片。

### 圖片轉影片

上傳圖片作為影片的起始幀或結束幀：

- **起始幀**（必填）：影片的第一個畫面
- **結束幀**（選填）：影片的最後一個畫面
- **循環影片**：若只上傳起始幀，可選擇生成循環影片

### 參考圖生成

上傳參考圖片來引導影片生成：

- **主體參考**：讓影片中的物件/角色與參考圖相似
- **風格參考**：讓影片風格與參考圖相似
- 點擊圖片上的標籤可切換參考類型
- 最多 3 張參考圖片

::: warning 限制
使用參考圖生成時，模型固定為 Veo 3.1，長度固定為 8 秒。解析度和比例可自由選擇。
:::

### 延伸影片

選擇歷史記錄中的影片，生成延伸內容。

::: info 注意
延伸影片的限制：
- 僅支援 720p 解析度的影片作為延伸來源
- 延伸長度固定為 8 秒
:::

## 影片參數

### 模型

- **Veo 3.1 Fast**：較快生成，品質稍低
- **Veo 3.1**：高品質，需要更多時間

### 影片長度

- **4 秒**：短片段，適合 GIF 式用途
- **6 秒**：中等長度
- **8 秒**：標準長度

### 解析度

- **720p**：較快生成，檔案較小
- **1080p**：高畫質
- **4K**：超高畫質，需要更多時間

### 影片比例

- **16:9**：標準橫式
- **9:16**：直式（適合手機）

## 提示詞技巧

### 描述動作

影片需要描述**動態**場景：

```
一隻蝴蝶緩緩從花朵上飛起，
陽光透過翅膀形成彩虹光暈，
背景是模糊的花園
```

### 鏡頭語言

使用電影術語可以獲得更好的效果：

```
慢動作特寫：水滴從葉子滴落
平移鏡頭：穿過森林的小路
推軌鏡頭：緩慢接近古老的城堡
```

### 風格指定

```
電影質感，淺景深，
溫暖的金色光線，
35mm 底片顆粒感
```

### 排除提示

影片模式獨有的「排除提示」功能，可以指定你不希望在影片中出現的元素：

```
卡通、繪畫、低品質、模糊、變形
```

## 音訊提示

Veo 3.1 支援生成影片音效。你可以在「音訊提示」分頁中設定三種類型的聲音：

### 對話

讓影片中的角色說出特定內容。使用**引號**標示要說出的語音：

```
「這一定是鑰匙，」他低聲說道
```

::: tip 提示
使用引號標示特定語音，模型會嘗試讓角色說出引號內的文字。
:::

### 音效

描述影片中的具體聲音效果：

```
輪胎尖銳的摩擦聲、引擎轟隆聲
```

### 環境音

設定場景的背景音：

```
背景中傳來微弱的詭異嗡嗡聲
```

::: warning 關於音訊生成
Gemini API（本應用程式使用的 API）**無法關閉音訊生成**，所有影片都會自動包含音效。若要生成完全靜音的影片，需要使用 Vertex AI API（需要 Node.js 後端環境，不支援純瀏覽器端）。

目前的音訊提示（對話、音效、環境音）僅用於引導 AI 生成特定類型的聲音，但無法完全禁用音訊。
:::

## 生成時間

影片生成通常需要 1-3 分鐘，取決於：

- 影片長度
- 解析度
- 內容複雜度
- API 負載

生成過程中可以做其他事情，完成後會收到通知。

## 預估費用

設定選項時，介面會即時顯示預估費用（每秒單價 × 影片長度），方便你在生成前了解成本。

實際費用請參考 [Google AI Pricing](https://ai.google.dev/gemini-api/docs/pricing)。

## 下載與分享

生成完成的影片可以：

- 下載 MP4 檔案
- 預覽播放
- 儲存到歷史記錄
